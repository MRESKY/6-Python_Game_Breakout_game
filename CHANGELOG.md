# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-28

### Added

- Initial release of Breakout Game
- Classic breakout gameplay with ball, paddle, and bricks
- Multiple levels with different brick patterns
- Score system with points for breaking bricks
- Lives system (3 lives, lose one when ball goes off screen)
- High score tracking with persistent storage
- Smooth paddle controls using arrow keys
- Collision detection system
- Game over screen with final score display
- Level progression with countdown timer
- Colorful brick types with different health values
- Clean object-oriented architecture
- Entity-component system design
- UI management system for HUD and game screens

### Game Features

- Ball physics with bounce mechanics
- Paddle movement with screen boundary detection
- Brick destruction with visual feedback
- Progressive difficulty across levels
- Heart emoji display for remaining lives
- Centered text rendering for game messages

### Technical Features

- Abstract base class for game objects
- Modular system architecture
- Clean separation of concerns
- Dependency injection support
- Collision detection abstraction
- Level management system
- Score persistence
